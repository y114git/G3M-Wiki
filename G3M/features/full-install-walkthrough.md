# Full Install Walkthrough

This step-by-step guide walks you through installing a free game using G3M's Full Install feature.

---

## Example: Installing DELTARUNE DEMO

### Step 1: Open Settings

Click the **Settings** tab in the tab bar (or the gear icon).

### Step 2: Go to Game Section

In the Settings sub-tabs, select the **Game** section.

### Step 3: Find DELTARUNE DEMO

Scroll to the DELTARUNE DEMO game entry.

### Step 4: Enable Full Install

Check the **Full Install** checkbox for DELTARUNE DEMO. You'll notice the main action button in the status bar changes from "Launch" to **Install**.

### Step 5: Click Install

Click the **Install** button in the status bar.

### Step 6: Choose Installation Folder

A folder picker dialog appears. Choose where you want the game to be installed. For example:

- `C:\Games\DELTARUNE_Demo`
- `D:\GameFiles\DELTARUNE_Demo`

Click **Select Folder**.

### Step 7: Wait for Download

G3M downloads the DELTARUNE DEMO files from the server. The progress bar shows:

- **Download progress** — Percentage of the file downloaded.
- **Status text** — "Downloading DELTARUNE DEMO..." or similar.

This may take several minutes depending on your internet speed.

### Step 8: Automatic Extraction

After download, G3M automatically extracts the game files into the selected folder. The progress bar updates to show extraction progress.

### Step 9: Installation Complete

Once extraction finishes:

- The game path is automatically set to the new folder.
- The Full Install checkbox is disabled (checked and grayed out — indicating the game is installed).
- The status bar shows "Installation complete" or similar.
- The action button reverts to **Launch**.

### Step 10: Launch the Game

You can now click **Launch** to start the DELTARUNE DEMO. No mods are needed — the game runs vanilla.

### Step 11: Install Mods (Optional)

Since DELTARUNE DEMO does not have a GameBanana ID, you cannot browse mods in the Mods Browser. Instead:

1. Obtain mods from other sources (community forums, Discord, direct downloads).
2. Import them via Library → Add Mod → Import Mod.
3. Or drag and drop mod files onto the G3M window.
4. Select the imported mod in the Library and click **Launch**.

## Example: Installing UNDERTALE Yellow

The process is identical:

1. Settings → Game → UNDERTALE Yellow → Enable **Full Install**.
2. Click **Install**.
3. Choose a folder.
4. Wait for download and extraction.
5. Done — the game path is set automatically.

For UNDERTALE Yellow, you **can** browse mods in the Mods Browser (GameBanana ID: 19606).

## Troubleshooting Full Install

| Problem | Solution |
| --- | --- |
| Install button is disabled | Make sure the Full Install checkbox is enabled. Make sure no other operation is in progress. |
| Download fails | Check your internet connection. The download server may be temporarily unavailable — try again later. |
| Extraction fails | Ensure there is enough disk space in the target folder. Check that the folder is writable. |
| Game won't launch after install | Verify the game executable exists in the installed folder. Check Settings → Game to confirm the path is set. |
| Full Install checkbox is missing | The game may not support Full Install (e.g., DELTARUNE full version, UNDERTALE, Pizza Tower require purchase). |
| Want to reinstall | Delete the game folder, clear the game path in Settings → Game, then re-enable Full Install. |
