# Analytics

G3M includes a two-tier anonymous analytics system to help the developer understand how the application is used.

---

## Two Tiers: Always-On and Opt-In

### Always-On

A small set of aggregate data is collected regardless of the opt-in setting. This data contains nothing that can identify you or your system — only general, non-personal usage signals such as the fact that the app was launched, that a game was started, or that a mod was installed. No usernames, file paths, IP addresses, or personal identifiers are included.

### Opt-In

Additional usage data is collected only when the user explicitly enables analytics. This provides the developer with more context about how G3M is being used (e.g., which features are popular, which games are most active).

**Opt-in is disabled by default.** To enable:

1. Go to Settings → General.
2. Check the **Anonymous analytics** checkbox.

Disabling opt-in at any time stops the collection of opt-in data immediately.

---

## Privacy

- No personally identifiable information is ever collected in either tier.
- No usernames, email addresses, IP addresses, file paths, or device identifiers are included.
- Data is sent securely to the G3M server over HTTPS.
