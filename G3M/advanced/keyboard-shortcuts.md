# Keyboard Shortcuts & Interactions

G3M supports various keyboard and mouse interactions for efficient navigation.

---

## Window Management

| Action | Shortcut / Gesture |
| --- | --- |
| Move window | Click and drag the title bar. |
| Resize window | Drag any window edge or corner. |
| Maximize / Restore | Double-click the title bar, or click the maximize button. |
| Minimize | Click the minimize button in the title bar. |
| Close | Click the close button in the title bar. Alt+F4 (Windows). |
| Fullscreen toggle | Settings → General → Fullscreen checkbox. |

---

## Navigation

| Action | Interaction |
| --- | --- |
| Switch tabs | Click a tab in the tab bar (Mods Browser, Library, Settings). |
| Switch game | Click the game selector dropdown in Library or Mods Browser. |
| Switch chapter (DELTARUNE) | Click a chapter tab in the Library. |
| Switch profile | Click the profile selector dropdown. |
| Switch settings sub-tab | Click a sub-tab within Settings. |

---

## Library Interactions

| Action | Interaction |
| --- | --- |
| Select/deselect a mod | Click the **Use** / **Unuse** button on a mod card. |
| Edit a mod | Click the **Edit** button on a mod card. |
| Delete a mod | Click the **Delete** button. Confirm in the dialog. |
| Export a mod | Click the **Export** button. |
| Open mod folder | Click the **Open Folder** button. |
| Open mod versions | Click the **Versions** button. |
| Search mods | Click the search button, type text, press Enter. |
| Clear search | Click the "×" button in the search field. |
| Toggle Direct Launch chapter | Double-click a chapter tab (DELTARUNE only). |
| Add a mod | Click the "+" button → Import Mod or Create Mod. |

---

## Mods Browser Interactions

| Action | Interaction |
| --- | --- |
| View mod details | Click a mod card to open the details overlay. |
| Close mod details | Click the back button, press Escape, or click outside the overlay. |
| Download a mod | Click **Download** in the mod details overlay. |
| Filter by tag | Click tag buttons in the filter bar (Customization, Gameplay, etc.). |
| Search mods | Click the search button, type text, press Enter. |
| Change sort | Click the sort dropdown and select an option. |
| Toggle sort direction | Click the sort direction button (arrow icon). |
| Toggle NSFW | Click the "Show NSFW" checkbox. |
| Load more mods | Scroll down to trigger pagination. |
| Right-click screenshot | Context menu: Open in browser, Copy image, Copy URL. |

---

## Modding Tools Interactions

| Action | Interaction |
| --- | --- |
| Switch tool tab | Click Convert DATA, Patch, Merge, Info, or Diff tab. |
| Select input file | Click the file selection button for each input. |
| Run operation | Click the action button (Convert, Apply, Merge, Inspect, Diff). |
| Cancel operation | Click **Cancel** if the operation is in progress. |

---

## Dialog Interactions

| Action | Interaction |
| --- | --- |
| Confirm / OK | Click the **OK** or **Confirm** button. Enter key in some dialogs. |
| Cancel / Close | Click the **Cancel** or **Close** button. Escape key. |
| Browse for file | Click the file/folder selection button. |

---

## Drag and Drop

| Source | Target | Result |
| --- | --- | --- |
| `.zip` / `.7z` / `.rar` file | Main window | Import as mod. |
| `.g3mpatch` / `.xdelta` / `.csx` file | Main window | Import as mod. |
| `.win` / `.ios` / `.unx` file | Main window | Import as mod. |
| Folder | Main window | Import if contains mod files. |
| URL | Main window | Download and import. |
| `.zip` file | Mod Versions dialog | Import as version snapshot. |

---

## Mouse Interactions

| Action | Interaction |
| --- | --- |
| Click mod card (Browser) | Opens mod details overlay. |
| Click mod card (Library) | Selects/highlights the mod (shows actions). |
| Right-click mod card | Context menu with additional options. |
| Right-click screenshot | Image context menu (open, copy, copy URL). |
| Click logo | Opens About dialog. |
| Click social button | Opens link in browser. |
| Hover over element | Shows tooltip (if available). |
| Scroll mod grid | Loads more mods (Browser pagination). |
| Scroll mod list | Scrolls through installed mods (Library). |

## Color Picker

When selecting a custom theme color:

1. Click the color button next to a color channel.
2. The system color dialog opens.
3. Pick a color using the color wheel, sliders, or hex input.
4. Click **OK** to apply.
5. The UI updates immediately with the new color.
